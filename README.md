# Amazon Clone

A frontend clone of the Amazon homepage built using **HTML5** and **CSS3**. This project was created to practice core web development concepts, including page layout, styling, Flexbox, and CSS Grid.

## Features

- Amazon-inspired homepage layout
- Navigation bar with search section
- Hero section
- Product cards
- Multi-column footer
- Hover effects
- Flexbox and CSS Grid layouts
- Font Awesome icons

## Technologies Used

- HTML5
- CSS3
- Font Awesome
